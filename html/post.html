Arrumando a Mesa

Bem, antes de mais nada precisamos arrumar a mesa. Vamos deixar o computador escolhido para o trabalho o mais leve possível.

Programas pesados e que alteram as bibliotecas do Windows (DLL) são os mais nocivos para o nosso empreendimento. Então, não tenha pena: Jogos, editores de vídeo, editores de imagens (principalmente os vetoriais), Office, emuladores de console de jogo, programas que consomem banda de internet tal como os P2P (Emule e companhia). Estes são alguns exemplos e devem ser eliminados se for possível. O melhor mesmo é uma instalação nova do Windows onde estes nunca tenham sido instalados e eventualmente modificado ou danificado o sistema.
